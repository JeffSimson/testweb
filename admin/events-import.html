Bulk Event Import

Admin-only helper for building your events.json file faster.

1. Paste Events

This tool does not publish automatically. It creates the correct JSON for you. Copy the result into content/events.json in GitHub, then commit.

Paste events in the simple format below. Separate events with a blank line.

Title: USSSA Northeast Opener
Category: Baseball
Date: April 24-26, 2026
Location: 515 Monmouth Road, Building 210, Jackson, NJ
Price: $695/team
Register: https://example.com
Image: /uploads/site/hero-aerial.jpg
Description: Weekend baseball tournament at Adventure Sports.
Featured: yes
Back to CMS

2. Copy Result

Copy this result into GitHub at content/events.json. Replace the whole file, then commit changes. 

// Your generated events.json will appear here.
Copied!

Valid categories: Baseball, Softball, Flag Football, Private Rentals, Camps, Tournaments.
